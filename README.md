## Atuação profissional

Atualmente participo do desenvolvimento de uma **plataforma corporativa de grande porte**, com backend desenvolvido em **Go**, baseada em **arquitetura de microsserviços e orquestração em Kubernetes**.

O projeto adota práticas modernas de engenharia, incluindo:

- Clean Architecture e organização em camadas
- Observabilidade com OpenTelemetry
- Autorização baseada em políticas com Casbin
- Armazenamento de objetos com MinIO
- Persistência em PostgreSQL
- Automação de pipelines de CI/CD
- Integração entre módulos e serviços

A solução possui visão estratégica de evolução para um **ERP moderno, altamente escalável e orientado a integração de domínios de negócio críticos**.

Em paralelo, atuo em **outros projetos institucionais internos**, trabalhando em ambientes multi-stack e contextos tecnológicos distintos:

- Desenvolvimento e manutenção de **APIs corporativas em Java**, utilizando **Spring Boot e Quarkus**
- Evolução de sistemas **C# com .NET / ASP.NET Core**
- Desenvolvimento de interfaces modernas com **TypeScript, React e Next.js**
- Participação em projeto utilizando **Python com Flask**, voltado para **integração com serviços de biometria facial e validação de identidade**, incluindo processamento de imagens e comunicação com APIs especializadas
- Integração de sistemas legados, padronização de ambientes e melhoria contínua dos fluxos de entrega

Essa atuação simultânea fortalece a visão de engenharia voltada à **escalabilidade, confiabilidade, interoperabilidade e sustentação de sistemas corporativos**.
